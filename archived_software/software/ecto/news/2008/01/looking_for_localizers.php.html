Looking for localizers

announcements / ecto3 / localization / release

ecto3 has been in beta for a while, even though it's been release mode stable for quite a few versions. It's time to actually take it out of beta, so I'm starting to work on localization. I already have people doing Chinese, Taiwanese, Danish, Finnish, and Spanish. I'm looking for more, incl. 日本語. If you're interested, drop me a line using the contact form.

Update: Localization offers received so far for languages:

  • German
  • French
  • Japanese
  • Portuguese
  • Swedish

Thanks much all!
