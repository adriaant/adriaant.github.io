Warcraft plugin

 

The World of Warcraft website does not have a feed. I wrote a little endo plugin that will create feed with the latest news on World of Warcraft. Download Warcraft.zip, unpack and double-click the Warcraft.endoplug file. It should automatically install in the proper location. Once you install it, just hit cmd-N and choose "Warcraft" from the pop-up menu.

Source is included for those interested in examples of endo plugins.

This plugin has been updated for the newest World of Warcraft website.
